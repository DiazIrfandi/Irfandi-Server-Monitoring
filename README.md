# Irfandi Server Monitoring

A modern Docker management and server monitoring dashboard built with React, Fastify, and Docker.

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Node](https://img.shields.io/badge/node-%3E%3D20-green)
![Docker](https://img.shields.io/badge/docker-supported-blue)

---

## ✨ Features

### 📊 System Monitoring

- CPU Usage
- Memory Usage
- Disk Usage
- Network Statistics
- System Information
- Real-time Dashboard

### 🐳 Docker Management

#### Containers

- List Containers
- Start Container
- Stop Container
- Restart Container
- View Container Information
- Create Container *(Coming Soon)*
- Delete Container *(Coming Soon)*

#### Images

- Pull Image
- Delete Image

#### Volumes

- View Volumes
- Delete Volume

#### Networks

- View Networks

### ⚙️ Settings

- Theme (Light / Dark / System)
- Screen Lock (PIN)
- More settings coming soon

## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/DiazIrfandi/Irfandi-Server-Monitoring.git
```

### Frontend

```bash
cd frontend

npm install

npm run dev
```

### Backend

```bash
cd backend

npm install

npm run dev
```
